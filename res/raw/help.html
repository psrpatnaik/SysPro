SysPro: An Android Mobile Application for Effective Teaching in System Programming

SysPro is a Mobile Application for System Programming subject of Third Year of Computer Science and Engineering. This subject contains six chapters namely

  • Language Processor
  • Assembler
  • Macro
  • Compilers
  • Linker
  • Loader

For each chapter, following contents are provided

  • Notes
  • PPT
  • Quiz
  • Question Bank
  • Games

For each chapter, Games part contains the following chapter related games

  • Word Scramble
  • Crosswords
  • Word Search
  • Word Matching

References:

  1. System Programming and Operating Systems -- 2nd Edition  D.M. Dhamdhere (TMGH) 
  2. System Programming  --  J. J. Donovan (Mc-Graw Hill)
